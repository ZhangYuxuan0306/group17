# 项目提纲

## 1. 做了什么？

**一个基于RAG的食品领域问答系统**

主要功能：

- 文档处理与向量库构建（基于pypdf, langchain, faiss，实现了读取->分块->建库->保存->加载->检索一体化流程）

-  ColBERTv2 精排（采用top-k策略，在faiss的余弦相似度检索基础上，基于colbert实现对检索文档的进一步排序（Reranked））
- 提示词嵌入的回答生成（将检索上下文嵌入提示词中，调用qwen3-14b模型输出带引用的答案，同时输出上下文命中率、延迟、吞吐指标）

-  RAGAS 自动评估（在忠实度，上下文精确率，回答相关性，召回率四个指标上打分，同时对于指标数值过低的情况实现失败案例分析与误差归因：（检索错误，重排错误，生成错误））
- web demo（网页界面可演示：提问、选择检索器、查看引用、执行评估……）
